# 💰 Finance App - Dashboard de Gestão Financeira

Este é um projeto Full Stack desenvolvido para gerenciar finanças pessoais, permitindo o controle de entradas, saídas e visualização de balanço em tempo real.

## 🚀 Tecnologias Utilizadas

- **Frontend:** Next.js 15, React, TypeScript, Tailwind CSS.
- **Backend:** Java Spring Boot.
- **Infraestrutura:** Docker & Docker Compose.
- **Ferramentas:** Turbopack, Git.

## 🛠️ Diferenciais Técnicos

- **Containerização Total:** Projeto configurado com Docker, garantindo que o ambiente de desenvolvimento seja idêntico ao de produção.
- **Tipagem Estrita:** Uso de TypeScript para garantir a integridade dos dados e evitar erros em runtime.
- **SSR & Turbopack:** Utilização das versões mais recentes do Next.js para máxima performance.
